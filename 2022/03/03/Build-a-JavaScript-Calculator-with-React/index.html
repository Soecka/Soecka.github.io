Build a JavaScript Calculator with React

Manta

2022-03-03

构建一个 JavaScript 计算器

Demo

设计要求

  1. 通过所有测试
  2. 添加额外的功能和视觉特色

初步想法

  1. 用React完成组件设计

  2. 运算逻辑有三种实现方法

    1. 按条件区分运算符，对前后两个输入进行运算
    2. 通过 replace() 清洗输入表达式，调用 eval() 进行计算[不推荐，eval() 较为危险，浏览器可能不支持]
    3. 多方查找后，找到 math.js 库，可以直接引用，更为安全

  3. 增加键盘监听事件，直接用键盘输入进行运算

解决步骤

草图设计

  1. UI 设计

  2. 各区域组件设计

    1. Display — 展示清洗过后的表达式
    2. Input — 展示上一个点击的按钮以及最后的运算结果
    3. Button Pad — 三类输入按钮
      1. 数字
      2. 运算符
      3. “AC” 及 “=” 键

逻辑实现

Button Pad

计划通过数组遍历生成按键，减少HTML代码量。keyCode用于监听 keydown 事件，id 用于区分操作符，value 用于传递值（ keyTrigger 非必要，可去除）。

button按键, 通过 props 传递 onClick行为 和 value

Button组件，添加键盘按键事件监听，继续向上一级组件传递 onClick 处理事件

Button Pad 组件，遍历向下传递数组数据，向上传递 pastClick 及 pastId 方法

App组件

state设计

expInput 为清洗后的输入表达式

display 用于展示输入

lastClick 用于展示按键输入

lastResult 为上一次计算结果

操作方法

  1. 处理输入，传递方法
  1. 处理操作符及清零
  1. 渲染各组件

CSS 美化

布局设计

  1. Display 区域为flex布局
  2. Button Pad 区域通过 Grid 模板布局

配色

  1. 直男粉+橄榄绿

    最终效果

  1. 在 Display 区域和 Button Pad 区域增加分割线
  2. 优化 displayHandle 的正则表达式，将多个合并为一个 match() 方法
